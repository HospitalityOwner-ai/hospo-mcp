Up and running in minutes

No developer needed. No custom code. Just drop in your API keys.

1

Install

pip install hospo-mcp — that's it. Works on Mac, Linux, Windows.

2

Configure

Add your API keys to .env. Or use mock mode to explore with realistic demo data first.

3

Connect

Add one JSON snippet to Claude Desktop or Cursor. Takes 60 seconds.

4

Ask

Talk to your AI like it knows your venue. Because now it does.

# claude_desktop_config.json
{
  "mcpServers": {
    "hospo-mcp": {
      "command": "hospo-mcp",
      "env": {
        "LIGHTSPEED_ACCESS_TOKEN": "your_token",
        "LIGHTSPEED_ACCOUNT_ID":   "12345",
        "XERO_ACCESS_TOKEN":       "your_token",
        "XERO_TENANT_ID":          "your_tenant",
        "DEPUTY_API_KEY":          "your_key",
        "DEPUTY_SUBDOMAIN":        "myvenue"
      }
    }
  }
}
